# 사무실 시설 안내

## 회의실 예약
- 그룹웨어 "예시 그룹웨어" → 시설 예약 메뉴
- 최대 2주 전부터 예약 가능
- 30분 단위로 예약, 최대 4시간 연속
- 예약 후 15분 내 미입실 시 자동 취소

## 회의실 종류
- 대회의실(20인): 본관 5층, 화상회의 장비 완비
- 중회의실(10인): 본관 3, 4층 각 2개씩
- 소회의실(4인): 각 층 2개, 즉석 사용 가능
- 폰부스(1인): 각 층 4개, 예약 없이 사용

## 사무용품
- 일반 사무용품: 각 층 비품실 자유 이용
- 노트북/모니터 등 IT 자산: IT 헬프데스크 신청
- 명함: 신규/재제작 시 인사팀에 신청, 1주 소요

## 출입 카드
- 분실 시 즉시 보안팀(내선 119)에 신고
- 재발급 수수료 1만원 (개인 부담)
- 임시 출입증은 1층 안내데스크에서 당일 발급
