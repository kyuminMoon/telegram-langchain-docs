# VPN 접속 가이드

## 접속 정보
- VPN 서버: vpn.example.internal
- 포트: 443 (TCP)
- 프로토콜: OpenVPN

## 신규 발급 절차
1. IT 헬프데스크에 신청 (헬프봇으로 티켓 생성 가능)
2. 본인 명의 사번과 휴대폰 번호 확인
3. 인증서(.ovpn 파일)와 임시 비밀번호를 회사 이메일로 수령
4. 첫 로그인 후 비밀번호 즉시 변경 필수

## 클라이언트 설치
- macOS: Tunnelblick 또는 OpenVPN Connect
- Windows: OpenVPN Connect
- iOS/Android: OpenVPN Connect (스토어 다운로드)

## 접속 안 될 때 체크리스트
1. 인증서 만료 확인 (90일마다 갱신 필요)
2. 회사 네트워크 외부에서 접속 중인지 확인
3. 로컬 방화벽이 443 포트를 막는지 확인
4. 그래도 안 되면 IT 헬프데스크 문의

## 재발급
- 인증서 만료 7일 전 자동 알림 메일 발송
- 분실 시 즉시 IT에 신고하여 기존 인증서 폐기 후 재발급
