# 휴가 정책

## 연차 부여 기준
- 신입사원: 입사 후 1개월 만근 시 1일씩 발생, 1년 미만 최대 11일
- 1년차 이상: 연 15일 기본 부여
- 3년차부터: 매 2년마다 1일씩 가산, 최대 25일

## 신청 절차
1. 사내 그룹웨어 "예시 그룹웨어"에서 휴가 신청서 작성
2. 직속 팀장 1차 결재
3. 인사팀 최종 승인
4. 승인 후 캘린더에 자동 등록

## 휴가 종류
- 연차: 위 기준에 따른 일반 휴가
- 반차: 오전(09:00~13:00) 또는 오후(13:00~18:00)
- 경조사 휴가: 결혼 5일, 직계가족 사망 5일, 형제자매 결혼 1일
- 병가: 연 5일, 진단서 제출 시 추가 가능

## 휴가 사용 제한
- 연속 5영업일 이상 사용 시 1개월 전 사전 공지 필요
- 분기 마감일(3/31, 6/30, 9/30, 12/31) 전후 1주는 사용 불가
