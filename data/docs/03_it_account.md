# 계정 관리 가이드

## 비밀번호 정책
- 최소 12자리, 영문 대소문자+숫자+특수문자 조합
- 90일마다 변경 권고
- 직전 5회 사용 비밀번호 재사용 불가

## 계정 잠금 해제
- 비밀번호 5회 연속 오류 시 30분 자동 잠금
- 즉시 해제 필요 시 IT 헬프데스크 문의
- 본인 인증(사번 + 휴대폰 SMS) 후 해제

## 권한 신청
- 시스템별 접근 권한은 본인 명의로 신청
- 팀장 승인 → 보안팀 검토 → IT 적용 (영업일 기준 2일 소요)
- 긴급 권한은 별도 결재 라인

## 퇴사 시 계정 처리
- 퇴사일 기준 즉시 모든 시스템 접근 차단
- 메일 데이터는 30일간 보관 후 영구 삭제
- 인수인계 자료는 별도 공유 드라이브에 사전 이관
